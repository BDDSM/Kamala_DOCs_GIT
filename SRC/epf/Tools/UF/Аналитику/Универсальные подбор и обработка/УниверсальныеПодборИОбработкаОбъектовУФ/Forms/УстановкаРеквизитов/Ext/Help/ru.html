Обработка "Установка реквизитов".

Выполняет установку (замену) реквизитов шапки объекта, изменение части времени объекта, строк табличной части или изменение записей регистров сведений. При выборе опции "Произвольная дата" у выбранных объектов изменяется только часть даты, соответственно при выборе "Произвольное время" часть даты не меняется.

Для табличных частей предусмотрены режимы обработки:

  1. Замена значений - заменяются значения отмеченных реквизитов в существующих строках.  Если выбран вид сравнения и значение для сравнения с текущим, замена будет производиться согласно выбранному виду сравнения.
  2. Добавление строки - поиск строк по значениям отмеченных реквизитов и добавление новой строки в случае отсутствия строк по заданному отбору. 
  3. Удаление строк по выбранным полям - поиск и удаление строк по заданному отбору.

Для выполнения установки реквизитов необходимо в табличном поле списка реквизитов в колонке "Значение" выбрать новые значения соответствующих реквизитов, включить реквизиты в обработку, изменив состояние флажка в строках в состояние "включено" и нажать кнопку "Выполнить" (клавиша F9). Установка реквизитов будет выполнен последовательно для каждого объекта выборки, включенного в обработку.
