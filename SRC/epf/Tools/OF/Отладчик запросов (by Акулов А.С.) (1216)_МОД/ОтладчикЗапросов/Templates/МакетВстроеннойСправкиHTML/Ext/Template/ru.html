Разработка запроса
Помимо стандартных возможностей консоли запросов (конструктор запроса, проверка синтаксиса), добавлены новые функции:
- Визуальная структура запроса
- Сделать вложенным запросом
- Извлечь запрос из вложенного
- Преобразовать вложенные запросы в пакет запросов
- Преобразовать пакетный запрос во вложенный

Анализ результатов выполнения запроса
Иногда нужно разобраться, почему запрос дал тот или иной результат.
Для этого, на вкладке "Результат" добавлены функции:
- просмотр виртуальных таблиц (ВТ)
- просмотр вложенных запросов (меню кнопки "Анализ")

Особые режимы выполнения пакетного запроса
Две кнопки в главном меню контролируют выполнение пакетного запроса:
- Не удалять временные таблицы запроса (при этом в пакетном запросе игнорируются удаления виртуальных таблиц);
- Пошаговое выполнение пакета запросов;

Шаблоны кода для отчетов и обработок
На странице "Запрос" есть кнопка "Код запроса", которая открывает форму с шаблонами кода для разработки отччетов.
В том числе, содержит вкладку "Модуль универсального отчета", с помощью которой можно разработать качественный универсальный отчет.
Рекомендуется использовать Универсальный отчет от alexk-is http://infostart.ru/public/84642/

Открыть запрос из отладчика
Существует возможность открыть запрос из конфигуратора (в режиме отладки). Подробнее в меню: "Файл" - "Открыть из отладчика" и "загрузть пакет запросов".
